## Slacker

Demo app for experimenting with RethinkDB's Horizon

## Setup (using Docker and Docker Compose)

##### Requirements

- Docker
- Docker Compose

Clone the Repo

```sh
git clone git@github.com:catkins/slacker.git
cd slacker
```

Fire it up!

```sh
docker-compose up
```

##### Local

##### Requirements

- NodeJS 4 or greater
- RethinkDB

Install the horizon CLI

```sh
npm install -g horizon@0.2.5-1
```

Fire it up!

```
hz serve --dev
```
